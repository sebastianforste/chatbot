# GC Copilot glossary

- **Matter**: A legal request or issue tracked end-to-end (intake -> decision -> evidence).
- **Decision log**: The artifact where counsel records rationale, risks, and controls applied.
- **GraphRAG**: Retrieval pipeline that turns uploaded regulations/contracts/policies into a graph index and supports retrieval/citations.
- **Guardrails**: Soft constraints on the assistant output (style, escalation rules, PII rules) enforced in prompts/flows.
- **Evidence**: Links to policies, docs, or approvals that support an answer.
