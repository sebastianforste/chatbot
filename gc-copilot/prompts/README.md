# GC Copilot prompts

This folder contains prompt templates tuned for in-house counsel workflows.

## Templates
- product-counsel.md
- regulatory-analysis.md
- contract-review.md

## Output expectations
Each prompt is designed to keep:
- concise executive summaries
- explicit assumptions and open questions
- a structured handoff section for the matter workspace

## Tips
Pick one prompt per conversation, copy into your assistant/system prompt, and ask for citations where possible. Keep decisions recorded in the workflow decision log.
